# UX Operativa

## Objetivo

Diseñar una interfaz tipo centro de monitoreo operativo.

## Requisitos

- dashboard superior fijo;
- tabla compacta;
- badges de estado;
- colores por prioridad;
- filtros sticky;
- búsqueda instantánea;
- desktop-first;
- responsive secundario;
- foco operativo y no comercial.

## Componentes

- KPIs;
- tabla operativa;
- filtros;
- búsqueda;
- acciones sugeridas.
